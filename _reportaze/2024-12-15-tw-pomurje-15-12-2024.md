Time Wizard
Format: Goat (april – oktober 2005)
Kraj: Okrepčevalnica Naci Črenšovci
Datum: 15. 12. 2024
Št. igralcev: 21
Struktura turnirja: Swiss (5 rund) + Top cut (Top 8)

Reprezentativnost deckov
Na turnirju je bilo moč najti 13 različnih strategij. Strategije, ki niso priznane, kot meta relevantne, so štete
pod »drugi«. Tukaj so štete Return varianta s Strike Ninjo, Fire Aggro in Pyramid of Light/Sphinx Control.
| Vrsta decka | Reprezentativnost dekov | Top 8 | Top 4 | Finale |
| --- | --- | --- | --- | --- |
| Chaos Turbo | 8 | 4 | 2 | 1 |
| Cat Burn | 2 | 2 | 1 | 1 |
| Chaos Control | 1 | 1 | 1 |  |
| Earth Aggro | 1 | 1 |  |  |
| Warrior | 2 |  |  |  |
| Reasongate | 1 |  |  |  |
| Goat Control | 1 |  |  |  |
| Soul Control | 1 |  |  |  |
| P.A.C.M.A.N. | 1 |  |  |  |
| Drugi | 3 |  |  |  |
| Skupaj | 21 | 8 | 4 | 2 |

Kot pričakovano, je bil daleč najpopularnejši deck Chaos Turbo, sicer s podpovprečno zastopanostjo glede na
dokazano uspešnost (le 38 %). Njegova reprezentativnost se je v Top Cutu malenkost povečala, nato pa ohra-
njala vse do finala (povsod 50 %). Zastopanost »aggro« strategij je bila precej nižja – mačje variante, Warrior
in Earth Aggro skupaj predstavljajo le 23 %. Cat Burn je ohranil 100% konverzno razmerje do Top 8 in potem
tudi bil prisoten skoz do finala, ki ga je tudi odnesel. Ostale strategije na žalost niso dosegle Top Cuta, razen
Chaos Control in Earth Aggro.

Od kod so prišli igralci
Če razdelimo prisotne v 4 večje regije, so se turnirja udeležili igralci iz Posavja, Maribora (in okolice), Ljubljane
in v največji meri iz Pomurja.
| Regija | Reprezentativnost regije | Top 8 | Top 4 | Finale |
| --- | --- | --- | --- | --- |
| Posavje | 4 | 4 | 3 | 2 |
| Maribor | 6 | 1 | 1 |  |
| Pomurje | 10 | 3 |  |  |
| Ljubljana | 1 |  |  |  |
| Skupaj | 21 | 8 | 4 | 2 |

Praktično polovica igralcev izhaja iz Prekmurja (48 %), medtem ko je njihova konverzija v Top 8 upadla (38 %),
medtem ko so bili popolnoma izločeni iz Top 4. Pričakovano na drugem mestu je bil kot regija zastopan Ma-
ribor z okolico (29 %), ki je svojo reprezentativnost izjemno zmanjšal že do Top 8. Posavje je predstavljalo le
19 % vseh igralcev, a so se vsi obdržali do Top Cuta in ohranili izjemno visoko konverzijo tako v Top 4 (75 %)
kot do finala, ki je bil popolnoma Brežiški. Ljubljano je zastopal 1 sam osamljen igralec brez sreče, ampak ne
dvomimo, da bo se bo konkurenca iz Ljubljane kmalu povečala.
